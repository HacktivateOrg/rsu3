Terminal

In this challenge you will:

  • connect to a remote server,
  • make a change to a webpage hosted there,
  • and confirm the change through a web browser.

Follow these steps:

  1. Visit this link in a new browser window:
    http://rsu3.hacktivate.org/hackers.html
  2. Open the terminal.
  3. Connect to remote webserver:
    ssh hacker@hacktivate.org
  4. Move to directory which hosts the webpage for our hacker club:
    cd /var/www/rsu3
  5. Open an editor to change the file:
    edit hackers.html
  6. Make a change to the page. Add your hacker details. Have fun with it. We're all friends here.
