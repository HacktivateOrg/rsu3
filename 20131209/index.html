Terminal

In this challenge you will:

  • connect to a remote server,
  • make a change to a webpage hosted there,
  • and confirm the change through a web browser.

Follow these steps:

  1. Connect to remote webserver:
    ssh hacker@hacktivate.org
  2. Move to directory which hosts the webpage for our hacker club:
    cd /var/www/rsu3
  3. Open an editor to change the file:
    edit index.html
